## Integrations

New Slack integration for instant alerts:

<CodeGroup tabs="JavaScript,Python">
```javascript
const slackData = {
  webhookUrl: "https://hooks.slack.com/services/YOUR_WEBHOOK",
  message: "Server downtime detected"
};
await fetch(slackData.webhookUrl, {
  method: "POST",
  body: JSON.stringify({ text: slackData.message })
});
```

```python
import requests
webhook_url = "https://hooks.slack.com/services/YOUR_WEBHOOK"
payload = {"text": "Server downtime detected"}
requests.post(webhook_url, json=payload)
```
